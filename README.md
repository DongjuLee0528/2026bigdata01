# 2026 Daelim University – Big Data Programming 01

##  Course Overview
This course focuses on learning the **Python programming language for data analysis software development**.  
Students will study Python from basic to advanced syntax and develop the ability to implement data analysis programs using various libraries.

- **Course Title**: Big Data Programming 01  
- **Semester**: Spring 2026  
- **University**: Daelim University  
- **Language**: Python  
- **Class Hours**: 3 hours per week  

---

##  Course Objectives
- Understand and apply **advanced Python syntax** required for data analysis software development.
- Develop the ability to implement **data analysis applications using Python features and libraries**.
- Strengthen practical skills in **real-world data processing and analysis** through hands-on exercises.

---

##  Course Contents
- Implementation of Python basic syntax  
- Implementation of Python advanced syntax  
- Definition and concepts of Big Data  
- Basic data analysis using Python  
- Advanced data analysis using Python
